Ein einfaches WordPress-Plugin zum Verwalten von Projekten und Aufgaben. Integriert in BuddyPress-Gruppen für beste Zusammenarbeit.

== Description ==

Ein Plugin für BuddyPress, mit dem Du Deine Projekte verwalten und jedem Mitglied einer bestimmten Gruppe eine Aufgabe zuweisen kannst. Du kannst die Priorität für jede Aufgabe festlegen ( Normal , Hoch und Kritisch ). BP Projektmanagement sendet auch E-Mail-Benachrichtigungen an Mitglieder, wenn neue Aufgaben verfügbar sind oder es neue Updates gibt.

BUDDYPRESS GROUP PROJEKT- UND AUFGABENMANAGEMENT
BP Projektmanagement ist ein kleines Projektmanagement-Plugin für WordPress. Dieses Projektmanagement-Plugin wurde als Add-On für BuddyPress erstellt.
